\chapter{CNC.js}
\label{ch:cncjs}
\section*{Instalación en Windows}
\begin{codeblock}{powershell}
npm install -g cncjs
cncjs --port 8000
\end{codeblock}

\section*{Conexión}
Abrir \texttt{http://localhost:8000}, seleccionar COM, baud 115200.
